## 🔍 주요 발견 사항

### 1. 메리츠 엑셀의 하이브리드 계산 방식

기존 가정: `residual_value = (vehicle_price - down_payment) × residual_rate`

**실제 방식:**
```
잔존가치 = 차량가 × 잔존율          (세금/수수료 제외)
금융대상 = 취득원가 - 선납금          (세금/수수료 포함)
감가상각 = 금융대상 - 잔존가치
```

**비즈니스 로직:**
- 잔존가치는 순수 차량 가치만 반영 (세금은 차량 가치가 아니므로)
- 하지만 고객은 취득원가 전체를 금융으로 조달

### 1.1 장기 계약 추가 프리미엄 (2025-11-06 업데이트)

**발견 사항:**
- 특정 차량은 48/60개월 장기 계약 시 추가 프리미엄 적용
- 엑셀 P열("고잔가추가 15,000"), Q열("고잔가추가1 10,000")에서 관리
- 차량별로 0~2%p 추가 프리미엄 차등 적용
- **중요**: 등급에 따라 P+Q 적용 비율이 다름 (등급별 계수)

**적용 로직:**
```
고잔가 = 일반잔가 + 기본_프리미엄 + (장기_프리미엄 × 등급_계수)

기본_프리미엄 (모든 기간):
  - APS/AJ: +8%p
  - VGS: +6%p

장기_프리미엄 (48/60개월만):
  - (P열 + Q열) × 등급_계수

등급_계수 (APS/West 기준):
  - High Tier (SA1, SA, A1, A): 1.0 (100% 적용)
  - Mid Tier (B, C, D, E, F, G, H): 0.75 (75% 적용)
  - Low Tier (I 이하): 0.5 (50% 적용)
```

**실제 사례:**
```
BMW 520i (APS SA1, P+Q=0.02):
  - 36개월: 63.5% + 8% = 71.5% (기본 프리미엄만)
  - 48개월: 55.5% + 8% + 2.0% = 65.5% (100% 적용)
  - 60개월: 49.5% + 8% + 2.0% = 59.5% (100% 적용)

BMW 740i (APS F, P+Q=0.02):
  - 36개월: 57.0% + 8% = 65.0% (기본 프리미엄만)
  - 48개월: 49.0% + 8% + 1.5% = 58.5% (75% 적용)
  - 60개월: 43.0% + 8% + 1.5% = 52.5% (75% 적용)

Mercedes GLC Coupe (APS B, P+Q=0.02):
  - 48개월: 53.0% + 8% + 1.5% = 62.5% (75% 적용)
  - 60개월: 47.0% + 8% + 1.5% = 56.5% (75% 적용)

GLB 250 4MATIC (P+Q=0):
  - 모든 기간: 기본 프리미엄 8%만 적용 (장기 프리미엄 없음)
```

### 2025-11-06 (2차): 등급별 장기 프리미엄 계수 추가
- **문제**: BMW 740i (F 등급) 등 일부 차량의 48/60개월 잔가가 0.5%p 차이
- **원인**: 등급에 따라 P+Q 적용 비율이 다름을 발견

  - F 등급: P+Q의 75% 적용 (1.5%p)
- **해결**:
  - `_get_grade_multiplier()` 메서드 추가
  - 등급별 계수 로직 구현:
    * High Tier (SA1, SA, A1, A): 100%
    * Mid Tier (B~H): 75%
    * Low Tier (I 이하): 50%
  - `_apply_premium_adjustment()` 메서드에 `grade` 파라미터 추가
  - 등급별 계수를 장기 프리미엄에 적용
- **검증**:
  - BMW 520i (SA1): 48개월 65.5% ✅
  - BMW 740i (F): 48개월 58.5% ✅
  - Mercedes GLC (B): 48개월 62.5% ✅
